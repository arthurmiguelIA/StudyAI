# 📚 StudyAI

> Plataforma fullstack de estudo com Inteligência Artificial para auxiliar estudantes no aprendizado, resolução de problemas e organização de estudos.

---

## 🚀 Sobre o Projeto

O **StudyAI** é uma aplicação que utiliza Inteligência Artificial para ajudar estudantes a aprender de forma mais eficiente, oferecendo suporte em diversas disciplinas com respostas inteligentes, explicações detalhadas e auxílio em tarefas acadêmicas.

A proposta do projeto é transformar a forma como os alunos estudam, trazendo uma experiência mais interativa, personalizada e acessível.

Plataformas semelhantes utilizam IA para fornecer explicações passo a passo, suporte em múltiplas disciplinas e aprendizado adaptativo ([App Store][1]).

---

## 🎯 Objetivos

* 📖 Facilitar o aprendizado com IA
* 🧠 Ajudar na compreensão (não só resposta)
* ⚡ Fornecer respostas rápidas e explicativas
* 🎓 Apoiar estudantes em diferentes níveis
* 🤖 Explorar integração de IA em aplicações reais

## 🛠️ Como Rodar o Projeto

### 1. Clone o repositório

```bash
git clone https://github.com/arthurmiguelIA/StudyAI.git
cd StudyAI
```

---

### 2. Backend

```bash
cd backend
python -m venv venv
source venv/bin/activate  # Linux/macOS
venv\Scripts\activate     # Windows

pip install -r requirements.txt
python app.py
```

---

### 3. Frontend

```bash
cd frontend
npm install
npm run dev
```
